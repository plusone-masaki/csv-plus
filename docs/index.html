CSV+

シンプルで使いやすい、タブCSVエディタ

ダウンロード →

複数OSに対応

Windows, MacOS, Linuxで利用できます。 CSV+さえあれば、対応OSで悩む必要はありません。

タブ方式を採用

CSVを開くたびにエディタを起動する時代はもう終わりです。 CSV+は複数のCSVを1つの画面にまとめます。

軽量で洗練された機能

複雑な操作や余分な機能を削ぎ落とし、 誰にとっても使いやすく軽快なエディタを目指しました。

# タブで複数のCSVを一元管理

ファイルのドラッグ＆ドロップにも対応し、 複数のファイルをまとめて開けます。



# 文字コード・改行コードの変換に対応

ShiftJIS や UTF-8 をはじめとした、 様々なエンコーディングに対応。 改行コードの変更も簡単に行なえます。
